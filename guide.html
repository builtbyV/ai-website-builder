Step 1: Opening Your Command Line (Terminal)

Think of the Terminal like a text message conversation with your computer. Instead of clicking buttons, you type commands. It's actually pretty cool once you get used to it!

Remember: Terminal, Command Line, and Console all mean the same thing - a place where you type commands.

Opening Terminal on Mac

Fastest Method: Use Spotlight Search!

  1. 1. Press Command ‚åò + Space at the same time
  2. 2. Type "Terminal" in the search box that appears
  3. 3. Press Enter when Terminal appears in the results

[Screenshot: Spotlight search with "Terminal" typed and Terminal.app highlighted]

What You Should See:

YourName@MacBook ~ %

Don't worry if yours looks slightly different. The important part is the % or $ at the end - that means it's ready for your commands!

Setting Up Git Bash on Windows

Important: Windows requires an extra step - installing Git Bash. This gives us the tools we need.

Step 1: Install Git for Windows

  1. 1. Go to git-scm.com/download/win
  2. 2. Download will start automatically
  3. 3. Run the installer - click "Next" for most options
  4. CRITICAL: On the "Adjusting your PATH" screen, choose the second option: "Git from the command line and also from 3rd-party software"

Step 2: Open Git Bash

  1. 1. Go to your Desktop (or any folder)
  2. 2. Right-click on empty space
  3. 3. Select "Git Bash Here"

What You Should See:

YourName@COMPUTER MINGW64 ~/Desktop
$

Step 2: Getting the Project Files

What's a Repository? It's just a fancy name for a project folder that remembers all changes. Think of it as a folder with a built-in time machine!

Downloading the Project

We'll use a command called git clone to download the AI Website Builder project.

1. Make sure you're in a good location (like Desktop):

cd ~/Desktop

2. Clone (download) the project:

git clone https://github.com/builtbyV/ai-website-builder.git

3. Move into the project folder:

cd ai-website-builder

Success looks like: You'll see some text about "Cloning into..." and then your prompt will change to show you're inside the ai-website-builder folder.

Verify You're in the Right Place

Type this command to see the project files:

ls

You should see files like README.md, setup.sh, index.html, etc.

Step 4: Setting Up Your AI Assistant

Now let's connect to your chosen AI assistant. Pick the tab for the one you installed:

Setting Up Google Gemini

Start Gemini by running:

npx gemini

The first time you run this:

  1. 1. It will show a URL or open your browser
  2. 2. Log in with your Google account
  3. 3. Authorize Gemini CLI
  4. 4. Return to terminal - you'll see it's ready

Note: Use a personal Google account for the free tier. Work/school accounts might not have access.

Success: You see a prompt and can type commands!

Setting Up Claude Code

Start Claude by running:

npx claude

The first time you run this:

  1. 1. It will open your web browser
  2. 2. Log in with your Anthropic account (requires Claude Pro subscription)
  3. 3. Authorize the connection
  4. 4. Return to your terminal - you'll see a > prompt

Test it by typing:

hello

Success: Claude responds with a friendly greeting!

Setting Up OpenAI Codex

Get Your API Key:

  1. 1. Go to platform.openai.com/api-keys
  2. 2. Log in or create an account
  3. 3. Click "Create new secret key"
  4. 4. Copy the key immediately (starts with sk-)

Set Your API Key:

export OPENAI_API_KEY="sk-your-key-here"

Start Codex:

npx codex

Success: You see a > prompt ready for commands!

Step 5: Installing GitHub Desktop

Why GitHub Desktop? It gives you a visual way to save your work and publish your website - no more command line needed for Git!

Installation

  1. 1. Download from desktop.github.com
  2. 2. Install it like any other app
  3. 3. Open GitHub Desktop
  4. 4. Click "Sign in to GitHub.com"
  5. 5. Complete the login in your browser

Add Your Project

  1. 1. In GitHub Desktop: File ‚Üí Add Local Repository
  2. 2. Click "Choose..." and select your ai-website-builder folder
  3. 3. Click "Add Repository"

Publish to GitHub

  1. 1. Click the "Publish repository" button

Note: This doesn't make your website live yet - it just backs up your code to GitHub. We'll make it live in the final step!

Step 7: Publishing Your Website! üöÄ

Time to make your website live on the internet!

Step 1: Save Your Changes

In GitHub Desktop:

  1. 1. You'll see your changed files listed
  2. 2. Write a short message like "Updated website content"
  3. 3. Click "Commit to main"
  4. 4. Click "Push origin"

Step 2: Enable GitHub Pages

  1. 1. Go to your repository on GitHub.com
  2. 2. Click "Settings" tab
  3. 3. Scroll down to "Pages" in the left sidebar
  4. 4. Under "Source", select "Deploy from a branch"
  5. 5. Under "Branch", select "main" and "/ (root)"
  6. 6. Click "Save"

Step 3: Wait for the Magic!

GitHub will build and deploy your site. This takes about 2-5 minutes.

Your website will be live at:

https://[your-username].github.io/ai-website-builder/

First-Time Note: You might see a 404 error at first. This is normal! Wait 5-10 minutes and refresh. Your site needs time to propagate across GitHub's servers.
