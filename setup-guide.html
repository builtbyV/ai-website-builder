Step 1: Opening Your Command Line (Terminal)

Think of the Terminal like a text message conversation with your computer. Instead of clicking buttons, you type commands. It's actually pretty cool once you get used to it!

Remember: Terminal, Command Line, and Console all mean the same thing - a place where you type commands.

Opening Terminal on Mac

Fastest Method: Use Spotlight Search!

  1. 1. Press Command ‚åò + Space at the same time
  2. 2. Type "Terminal" in the search box that appears

Setting Up Git Bash on Windows

Important: Windows requires an extra step - installing Git Bash. This gives us the tools we need.

Step 1: Install Git for Windows

  1. 1. Go to git-scm.com/download/win
  2. 2. Download will start automatically
  3. 3. Run the installer - click "Next" for most options
  4. CRITICAL: On the "Adjusting your PATH" screen, choose the second option: "Git from the command line and also from 3rd-party software"

Step 2: Open Git Bash

  1. 1. Go to your Desktop (or any folder)
  2. 2. Right-click on empty space
  3. 3. Select "Git Bash Here"

What You Should See:

YourName@COMPUTER MINGW64 ~/Desktop
$

Step 2: Getting the Project Files

What's a Repository? It's just a fancy name for a project folder that remembers all changes. Think of it as a folder with a built-in time machine!

Downloading the Project

We'll use a command called git clone to download the AI Website Builder project.

1. Make sure you're in a good location (like Desktop):

cd ~/Desktop

2. Clone (download) the project:

git clone https://github.com/builtbyV/ai-website-builder.git

3. Move into the project folder:

cd ai-website-builder

Success looks like: You'll see some text about "Cloning into..." and then your prompt will change to show you're inside the ai-website-builder folder.

Verify You're in the Right Place

Type this command to see the project files:

ls

You should see files like README.md, setup.sh, index.html, etc.

Step 5: Building Your Website!

The Two-Terminal Setup:

You need TWO terminal windows open:

  1. 1. Terminal 1: Runs the preview server (stays busy)
  2. 2. Terminal 2: Where you chat with AI

Start Your Preview Server (Terminal 1)

In your first terminal, run:

You'll see something like:
